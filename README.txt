Название программы: Рейтинговая система.

Функциональность:

Добавление пользователей.
Начисление баллов пользователям.
Определение лучшего пользователя.
Прогнозирование будущих баллов.

Требования:

Python 3.x
Tkinter
NumPy
Scikit-learn

Установка:

Установите Python 3.x на свой компьютер.
Установите Tkinter, NumPy и Scikit-learn с помощью pip: "pip install tk numpy scikit-learn".
Скачайте программу управления рейтингом пользователей (rating_system.py) и запустите ее с помощью Python: "python employee_app.py".

Использование:

Запустите программу управления рейингом пользователей.
Введите имя сотрудника и баллы в соответствующие поля.
Нажмите кнопку "Add Points" чтобы начислить баллы пользователю.
Программа автоматически определит лучшего пользователя и спрогнозирует будущие баллы.

Примечания:

Программа использует линейную регрессию для прогнозирования будущих баллов.
В реальном приложении можно использовать более сложные модели и данные.
Программа не сохраняет данные о пользователях и баллах между запусками.
